Use Case:

Title: Language Translation Accessability

Primary Actor: User 

Stakeholders and Interests:

User - Wants to translate posts to their language.
Poster - Wants to respond in native language and translate after. 

Precondition: User with granted access to read and respond to threads 

Trigger: Insert stuff 

Main success scenario:

  1. 1. User accesses forum
  2. 2. User sees default english language interface
  3. 3. User sees a language selection tool 
  4. 4. User changes interface to native language
  5. 5. Forum automatically recognizes native language responses
  6. 6. Forum offers 2-way translation for reading/writing threads/posts

Extensions

  • 4.a) Language not listed
    • The language desired may be too uncommon to be implemented at the time 
  • 5.a) Language not supported
    • The language is not listed and is therefore unsupported for recognization
  • 6.a) Language cannot be translated
    • The language is not supported and is therefore unable to be translated
