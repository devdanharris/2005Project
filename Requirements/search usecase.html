Title: Searching Facility
Primary Actor: User
Stakeholders and Interests: User - wants to be able to search for a topic or user.

Precondition: User with logged in already.
Trigger: User selects search button.
Main success scenario:

  1. User selects search button.
  2. SuD displays posts/people related to search.

Extensions:

  1. 2a. No matches on the keyword entered:
    1. 2a1. SuD notifies user on unsuccessful search.
    2. 2a2. User initiates Searching Facility use case
