Title: Creation of Discussion Groups and Optional Restriction of Topics to Created Groups
Primary Actor: User
Stakeholders and Interests: User - wants to be able to create a discussion group to share information privately among other users.

Precondition: User with logged in already.
Trigger: User selects create group button.
Main success scenario:

  1. User selects create group button.
  2. SuD asks user to create a channel password.
  3. User enters password.
  4. SuD notifies user of successful channel creation.

Extensions:

  1. 2a. User wants a public channel:
    1. 2a1. User clicks "No Password" button.
    2. 2a2. SuD notifies user of successful channel creation.
