Title: User Creation and Editing of Posts
Primary Actor: User
Stakeholders and Interests: User - trying to make a post on the SuD.
Precondition: User has access to SuD.
Trigger: User clicks add post or replies button.
Main success scenario:

  1. User logs ind.
  2. User clicks add post or replies button.
  3. SuD displays postings.
  4. SuD notifies poster of successful posting.

Extensions:

  1. 1a. User not registered for SuD:
    1. 3a1. User clicks add post or replies button .
  2. 4a. Poster wants to edit post:
    1. 4a1. Poster clicks edit button .
    2. 4a2. Poster corrects errors made.
    3. 4a3. Poster initiates User Creation and Editing of Posts use case
  3. 4b. Poster wants to delete thread:
    1. 4a1. User clicks delete button .
    2. 4a2. SuD notifies poster of successful removal.
